## 🚀 Super Quick Start

```bash
# 1. Initialize git
git init
git add .
git commit -m "Initial commit"

# 2. Push to GitHub (create repo first on github.com)
git remote add origin https://github.com/YOUR-USERNAME/gym-house.git
git push -u origin main

# 3. Deploy on Netlify
# Go to app.netlify.com → Import from GitHub
```

**Done!** 🎉
