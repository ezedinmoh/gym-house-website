# 🏋️ Gym House - Modern Fitness Website

A fully responsive fitness website with interactive calculators, workout planners, and modern UI/UX design.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

---

## 🌟 Features

### 📄 Pages
- **Home** - Hero section, features, testimonials
- **About** - Mission, vision, trainer profiles
- **Classes** - Filterable class schedule
- **Pricing** - 3 membership tiers
- **Blog** - Fitness tips and articles
- **Contact** - Contact form with Google Maps
- **Tools** - 6 fitness calculators
- **Workout Planner** - Pre-made workout routines
- **Dashboard** - User profile
- **Developer Info** - Professional contact page

### 🧮 Fitness Calculators
1. BMI Calculator
2. Calorie Calculator (TDEE)
3. Workout Timer
4. One Rep Max Calculator
5. Body Fat Calculator
6. Water Intake Calculator

### ✨ Interactive Features
- 🌙 Dark mode toggle
- 📊 Scroll progress indicator
- 🔍 Global search modal
- ⬆️ Back to top button
- ❤️ Favorites system
- 📱 Fully responsive design
- 🎨 Smooth animations

## 🛠️ Technologies

- HTML5, CSS3, JavaScript (ES6+)
- Font Awesome 6.4.0
- Unsplash API (images)
- Google Maps Embed API
- LocalStorage for preferences
